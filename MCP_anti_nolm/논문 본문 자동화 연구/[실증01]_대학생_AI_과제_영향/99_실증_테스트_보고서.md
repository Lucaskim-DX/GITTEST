# 논문 본문 자동화 연구: 제1차 실증 테스트 보고서

본 문서는 '학부 과제용 논문 본문 자동화 프레임워크'의 유효성을 검증하기 위해 진행된 제1차 실증 테스트 과정을 기록한 보고서입니다.

---

## 1. 실증 개요
- **실증 주제**: 척추 및 관련 연부조직 질환의 병태생리적 이해 (정의, 원인, 부위별 특성)
- **테스크 목표**: AI 모델의 기본 지식(Stage 1)과 전문 학술 데이터(Stage 2)의 결합을 통한 고품질 학술 리포트 자동 생성
- **사용 도구**: Antigravity(Agent), NotebookLM(Grounding Engine), DBpia/Google Search(Resource)

## 2. 실증 진행 과정

### [Stage 1] 신속 초안 생성 (Fast Drafting)
- **수행 내용**: 짧은 주제문과 요구사항 이미지를 분석하여 논리적 목차를 도출하고, AI의 내부 파라미터 지식을 활용하여 전체 뼈대를 작성함.
- **주요 성과**:
    - 연구의 배경, 목적, 주요 질환(디스크, 협착증)의 기본 정의를 포함한 1차 초안(`03_완성_초안.md`) 완성.
    - 전체 논문의 방향성과 논리 전개 구조를 5분 이내에 확정.
- **한계점**: 구체적인 임상 수치나 전문적인 병태생리 기전(에너지 위기 가설 등)의 서술 깊이가 부족함.

### [Stage 2] 근거 기반 심화 집필 (Source-Grounded Writing)
- **수행 내용**: 대한척추외과학회 및 전문 학술지에서 추출한 팩트 시트(`00_심화_소스_목록.md`)를 NotebookLM에 주입하고, 각 섹션을 딥다이브 형식으로 확장함.
- **주요 성과**:
    - **전문성 극대화**: 추간판 미세 구조 비율(수핵 50.7%, 섬유륜 41.3% 등)과 같은 정확한 데이터 이식.
    - **분량 확장**: 본론 섹션을 세분화하여 A4 3매 이상의 제출 규격 달성.
    - **정확도 향상**: 할루시네이션 없이 업로드된 소스 내의 전문 용어(Ligamentous hypertrophy, Sequestration 등)를 정확히 활용.

## 3. 실증 결과 비교 가치

| 구분 | Stage 1 (초안) | Stage 2 (심화본) |
| :--- | :--- | :--- |
| **정보의 깊이** | 일반적 정의 수준 | 전문 기전 및 수치 포함 |
| **정확성** | AI 추론에 의존 | 실제 학술 소스 기반 (Fact-checked) |
| **문서 분량** | 약 1.5 ~ 2매 | **3매 이상 (제출 규격 충족)** |
| **용도** | 방향성 검토용 | **최종 제출용** |

## 4. 최종 결론
본 실증을 통해 **'2단계 하이브리드 파이프라인'**이 학부 과제 수준을 넘어 전문적인 학술 리포트 생성에도 매우 유효함을 확인함. 
특히 NotebookLM을 '전문 지식 저장소'로 활용하고, AI 에이전트가 이를 '집필서 기반으로 인출'하는 구조는 할루시네이션 방지와 품질 고도화에 있어 최적의 조합임이 입증됨.
